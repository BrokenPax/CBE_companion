Cross Bronx NP Companion — GitHub Pages / Phone Bundle

This bundle includes the corrected NP PLAN helper:
- NP PLAN does not call Audit
- refresh an Organization if possible
- then record the next action

It also expands the EVENT helper to reflect the NP event-decision rules.

How to deploy:
1. Unzip this bundle.
2. Upload all files to the root of your GitHub Pages repo.
3. In GitHub: Settings → Pages → Deploy from branch → main → /(root).
4. Open the Pages URL on Android Chrome.
5. Use Chrome menu → Add to Home screen / Install app.

If the old version still appears:
- refresh twice
- or clear site data
- or remove and reinstall the home-screen app
because the service worker caches files.
