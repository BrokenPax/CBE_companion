Standard JavaScript GitHub bundle

This zip contains the current standard-JS site files:
- index.html
- app.js
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

To update GitHub Pages:
1. Upload/overwrite these files in the root of your repo.
2. Commit the changes.
3. Wait for Pages to redeploy.
4. If your phone still shows the old build, clear site data or reinstall the home-screen app because the service worker caches files.
